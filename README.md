Приложение разработано как серверная часть (API) сайта BlogMan на Node.js. 

### Run app
1. Create ```.env``` file in root folder
2. Pass there your keys:

    ```
    MONGO_URI=
    JWT_KEY=
    PORT=
    IMAGE_API_KEY=
    ```
3. Install dependencies:
    ```
    npm install
    ```
4. Run dev:
    ```
    npm run dev
    ```
